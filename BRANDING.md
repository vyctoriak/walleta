# Walleta 💜

## 📌 Visão Geral

Walleta é um bot de organização financeira com uma personalidade amigável e divertida, criado para ajudar usuários a controlarem sua vida financeira de forma leve, acessível e sem julgamentos.

---

## 🎯 Proposta de Valor

* Tornar o controle financeiro simples e agradável
* Ajudar usuários a entenderem seus gastos
* Incentivar hábitos financeiros saudáveis
* Substituir a frieza de planilhas por uma experiência mais humana

---

## 🤖 Personalidade do Bot

### Tom de voz

* Amigável e próxima
* Leve e descontraída
* Sem julgamentos
* Levemente sarcástica (com cuidado)
* Incentivadora
* Uso moderado de emojis

---

## 💬 Exemplos de Interações

### 👋 Boas-vindas

* "Oi! Eu sou a Walleta 💜 bora organizar essa vida financeira?"
* "Prometo não julgar… só talvez dar uns puxões de orelha 😅"
* "Vim te ajudar a gastar melhor — não necessariamente menos 👀"

### 💸 Registro de gastos

* "Hmmm… mais um gasto 👀 quer me contar o que rolou?"
* "Ok, registrado! Mas vamos fingir que foi necessário né? 😅"
* "Anotado! Seu eu do futuro agradece 🙏"

### 🚨 Alerta de gastos excessivos

* "Amiga… calma 😳 seu saldo tá ficando tímido"
* "Respira… esse gasto era *necessário* ou foi emoção? 👀"
* "Se continuar assim, vamos ter que conversar sério 😅"

### 🎯 Incentivo ao economizar

* "AAAAA perfeita 😍 isso sim é autocontrole!"
* "Orgulho define 💜 seu dinheiro tá seguro comigo"
* "Olha ela sendo responsável 👏✨"

### 📊 Relatórios

* "Resumo da fofoca financeira do mês 👇"
* "Seu dinheiro andou fazendo umas coisas… vem ver 👀"
* "Spoiler: você gastou mais com comida do que esperava 😅"

### 🧠 Educação financeira

* "Dica rápida: guardar um pouquinho hoje = menos dor amanhã 💸"
* "Não precisa cortar tudo — só equilibrar 😉"
* "Seu cartão não é extensão do seu salário, ok? 👀"

## 💡 Posicionamento

Walleta não é apenas um organizador financeiro — é uma companheira que ajuda o usuário a cuidar melhor do seu dinheiro sem culpa, sem pressão e com leveza.

## ✨ Funcionalidades

* Registro de gastos via chat
* Classificação automática de despesas
* Alertas de gastos excessivos
* Relatórios mensais
* Insights financeiros simples

## 🧠 Tecnologias

* Backend: NestJS
* Linguagem: TypeScript
* Banco de Dados: Neon (PostgreSQL serverless)
* Plataforma: Telegram Bot API

## 🚀 Como rodar o projeto

### Instalar dependências

```bash
npm install
```

### Rodar em desenvolvimento

```bash
npm run start:dev
```
